# Task Plan

## Goal
Audit the running Mac All You Need app end-to-end, capture all reachable pages, onboarding, permissions, popups, pills, and indicators, then improve the UI/flow so it feels modern, simple, smooth, and production-ready.

## Phases
1. Inventory live surfaces and capture screenshots.
2. Compare live UI with source code and identify gaps.
3. Implement the highest-impact UI/flow improvements.
4. Build and verify the changed surfaces.
5. Summarize remaining risks and next steps.

## Current Phase
Phase 3 in progress.

## In Scope Now
- Finish the highest-impact UI/flow polish on the live app surfaces we have already verified.
- Keep tightening setup, empty states, and direct-entry guidance so the app feels foolproof.
- Rebuild after each surgical change and avoid expanding scope into unrelated refactors.

## Notes
- User explicitly allowed accepting system prompts.
- Avoid destructive data resets unless explicitly confirmed.
- Keep changes surgical and tied to observed issues.
