Cleanup Review — Design Demo

Local rule-based review sheet. Nothing closes until the user selects items and taps Apply selected.

What Cleanup does

  • Scans open tabs/windows on device only (no AI).
  • Groups candidates: exact URL duplicates, stale tabs, empty windows, background apps.
  • Protects active, pinned, audible, and private tabs by default.
  • User confirms each batch — same review pattern as AI Organize.

Interaction

  • Groups with >5 items start collapsed.
  • Checkboxes default off.
  • Per-group toggle selects or clears the whole group.
  • Protected rows are disabled with a reason pill.
